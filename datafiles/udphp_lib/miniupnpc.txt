MiniUPnPc
---------

This folder contains binaries for MiniUPnPc. The license can be found in the LICENSE/LICENSE.txt file provided with this product.

The source code for MiniUPnPc can be found at: https://github.com/miniupnp/miniupnp

upnpc-static.exe is part of the oficial release 2015-09-18 for windows and was included in this official binary download:
    http://miniupnp.free.fr/files/download.php?file=upnpc-exe-win32-20150918.zip

upnpc-static is a linux 32bit static binary and is compiled by GMnet project maintainer Parakoopa using the 1.9. release:
    https://github.com/miniupnp/miniupnp/releases/tag/miniupnpc_1_9
